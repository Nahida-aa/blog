# github Discussions

## Categories

- Announcements
- General
- Ideas
- Polls
- Q&A
- Show & Tell : 用于展示项目和成就

## Discussion Format

- Open-ended discussion
  Enable your community to have conversations that don't require a definitive answer to a question. Great for sharing tips and tricks or just chatting.
- Question / Answer
  Enable your community to ask questions, suggest answers, and vote on the best suggested answer.
-  Announcement
  Share updates and news with your community. Only maintainers and admins can post new discussions in these categories, but anyone can comment and reply.
- Poll
  Gauge interest, vote, and interact with other community members using polls.

## Permissions

1. **Public**：任何人都可以查看和参与讨论。
2. **Private**：只有仓库的协作者可以查看和参与讨论。
3. **Read**：用户可以查看讨论，但不能参与。
4. **Write**：用户可以查看和参与讨论。

## API
```sh
https://api.github.com/repos
```

1. **获取讨论列表**：
   ```http
   GET /repos/{owner}/{repo}/discussions
   ```

2. **获取单个讨论**：
   ```http
   GET /repos/{owner}/{repo}/discussions/{discussion_number}
   ```

3. **创建讨论**：
   ```http
   POST /repos/{owner}/{repo}/discussions
   ```

4. **更新讨论**：
   ```http
   PATCH /repos/{owner}/{repo}/discussions/{discussion_number}
   ```

5. **删除讨论**：
   ```http
   DELETE /repos/{owner}/{repo}/discussions/{discussion_number}
   ```

6. **获取讨论评论**：
   ```http
   GET /repos/{owner}/{repo}/discussions/{discussion_number}/comments
   ```

7. **创建讨论评论**：
   ```http
   POST /repos/{owner}/{repo}/discussions/{discussion_number}/comments
   ```
