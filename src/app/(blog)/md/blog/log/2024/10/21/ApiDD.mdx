# API 文档

## 概述
本API文档详细描述了联合创作平台的API接口，包括用户管理、项目管理等功能。

## API 端点

### 用户注册
- **URL**：`/api/register`
- **方法**：POST
- **请求参数**：
  - `email`：用户邮箱（字符串，必填）
  - `username`：用户名（字符串，必填）
  - `password`：用户密码（字符串，必填）
- **响应**：
  - 成功：`{ "message": "注册成功" }`
  - 失败：`{ "error": "注册失败" }`
- **示例请求**：
  ```json
  {
    "email": "user@example.com",
    "username": "user123",
    "password": "password123"
  }
  ```
- **示例响应**：
  ```json
  {
    "message": "注册成功"
  }
  ```

### 用户登录
- **URL**：`/api/login`
- **方法**：POST
- **请求参数**：
  - `email`：用户邮箱（字符串，必填）
  - `password`：用户密码（字符串，必填）
- **响应**：
  - 成功：`{ "token": "JWT_TOKEN" }`
  - 失败：`{ "error": "登录失败" }`
- **示例请求**：
  ```json
  {
    "email": "user@example.com",
    "password": "password123"
  }
  ```
- **示例响应**：
  ```json
  {
    "token": "JWT_TOKEN"
  }
  ```

### 创建项目
- **URL**：`/api/projects`
- **方法**：POST
- **请求头**：
  - `Authorization`：Bearer JWT_TOKEN
- **请求参数**：
  - `name`：项目名称（字符串，必填）
  - `description`：项目描述（字符串，必填）
- **响应**：
  - 成功：`{ "message": "项目创建成功", "project": { "id": 1, "name": "项目名称", "description": "项目描述" } }`
  - 失败：`{ "error": "项目创建失败" }`
- **示例请求**：
  ```json
  {
    "name": "项目名称",
    "description": "项目描述"
  }
  ```
- **示例响应**：
  ```json
  {
    "message": "项目创建成功",
    "project": {
      "id": 1,
      "name": "项目名称",
      "description": "项目描述"
    }
  }
  ```

### 获取项目列表
- **URL**：`/api/projects`
- **方法**：GET
- **请求头**：
  - `Authorization`：Bearer JWT_TOKEN
- **响应**：
  - 成功：`{ "projects": [ { "id": 1, "name": "项目名称", "description": "项目描述" }, ... ] }`
  - 失败：`{ "error": "获取项目列表失败" }`
- **示例响应**：
  ```json
  {
    "projects": [
      {
        "id": 1,
        "name": "项目名称",
        "description": "项目描述"
      }
    ]
  }
  ```

### 获取项目详情
- **URL**：`/api/projects/{id}`
- **方法**：GET
- **请求头**：
  - `Authorization`：Bearer JWT_TOKEN
- **响应**：
  - 成功：`{ "project": { "id": 1, "name": "项目名称", "description": "项目描述" } }`
  - 失败：`{ "error": "获取项目详情失败" }`
- **示例响应**：
  ```json
  {
    "project": {
      "id": 1,
      "name": "项目名称",
      "description": "项目描述"
    }
  }
  ```
