# HTTP API
## XMLHttpRequest
## fetch
内部采用了 Promise 处理数据
```ts
fetch('https://api.github.com/users/Nahida-aa')
  .then(res => res.json())
  .then(data => console.log(data))
  .catch(error => console.error(error))
// or

async () => {
  try {
    const res = await fetch('https://api.github.com/users/Nahida-aa')
    const data = await res.json()
    console.log(data)
  } catch (error) {
    console.error(error)
  }
}
```
