## issues
### `GET /repos/{owner}/{repo}/issues`
```sh
GET https://api.github.com/repos/Nahida-aa/blog/issues
GET https://api.github.com/repos/Nahida-aa/blog/issues/{issue_number}
```
### `POST /repos/{owner}/{repo}/issues`
```sh
POST https://api.github.com/repos/Nahida-aa/blog/issues
Authorization

{
  "title": "Found a bug", # required string | int
  "body": "I'm having a problem with this.", # string 
  assignee : "Nahida-aa", # string | null
  "milestone": 1, # int | null | string
  "labels": [ # string[]
    "bug"
  ],
  "assignees": [ # array of strings
    "Nahida-aa"
  ]
}
```
### `PATCH /repos/{owner}/{repo}/issues/{issue_number}`
```sh
PATCH https://api.github.com/repos/Nahida-aa/blog/issues/1
Authorization

{
  "title": "Found a bug", #  string | int
  "body": "I'm having a problem with this.", # string | null
  "assignee" : "Nahida", # string | null
  "state": "open", # string, eg: open, closed
  "state_reason": "moved", # string | null, 如果状态没有改变，这个字段将被忽略
  "milestone": 1, # int | null | string
  "labels": [ # string[], 替换此 Issue 的标签s, 如果传入空数组，将清空所有标签
    "bug"
  ],
  "assignees": [ # array of strings
    "Nahida-aa"
  ]
}
```
