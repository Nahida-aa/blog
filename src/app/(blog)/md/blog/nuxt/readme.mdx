### Nuxt 3 Kitchen Sink Template
`Kitchen Sink` 意思是 除了厨房水槽什么都有，其他所有东西都在里面, 即应有尽有
```yml
assets/css/:
  - tailwind.css
components/:
  - content/:
    - CodeBlock.vue
  - SideNav.vue
  - ...
content/:
  - data-fetching.md
  - index.md
  - middleware.md
  - ...
layouts/:
  - default.vue
middleware/
    - auth.ts
pages/:
  - private.vue
  - [...slug].vue
public/
server/api/:
  - info.ts
  - hello.ts
  - count.ts
.npmrc
app.vue
nuxt.config.ts
package.json
tailwind.config.js
tsconfig.json
pnpm-lock.yaml
.gitignore
README.md
```
