- 身份证
- 支付功能: 银行卡验证
- ...
  可填入更多信息:
- 基础身份信息: 身高\性别\年龄\体重...
- 社区基础信息: 填写完整自动开通鉴赏者这个用户类型
  - 玩 mc 多久了
  - 为什么喜欢 mc
  - 喜欢 mc 什么内容:
    - 生存
    - 建筑
    - 服务器
    - 风格: [现代, 中世纪, 奇幻, 科幻, ...]
    - ...
  - 是否玩服务器,如果玩:
    - 模组服务器
    - 原版服务器
    - 是否愿意在服务器消费
    - 是否支持那些提供服务器的团队
    - ...
  - 是否购买正版, if yes: 输入正版 id
  - ...
    开通鉴赏者等于成为正式用户
