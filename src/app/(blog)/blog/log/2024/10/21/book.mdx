# mc 联合创作平台

## overview

## 项目背景
联合创作平台旨在通过互联网信用体系和社交网络的结合，帮助用户找到能够满足其需求的伙伴，实现价值互补。平台主要面向Minecraft内容创作者，提供一个在线合作和资源整合的环境。

<summary>来源和细节</summary>
<p>
大家都有一个共同的身份或一个标签，那就是学生，在这个身份下或这个标签情况下，大家有一个共同的需求。当然如果你想好好学习的话，那就是认识能帮到你学业的这种伙伴或同学，你们可以一起督促学习，共同进步，一起谈论你们共同的专业。在社交网络具备对用户的身份、需求、想法、性格、能力的一个划分的基础之后，在这样的社交产品之中，你是可以认识到可以一起谈论你们的专业，你们的理想，你们的学习内容的一个伙伴的。

## base features
1. **交互核心**：根据用户类型呈现不同内容，便捷高效地获取信息与服务。
2. **公告栏板块**：各用户类型下的公告展示。
3. **团队/项目动态宣传板块**：展示团队和项目的最新动态。
4. **用户中心板块**：用户的个人中心。
5. **项目宣传板块**：各用户类型下的项目宣传。
6. **我的团队板块**：用户的团队信息。

这个交互原型，我一共分成了五个板块。最上面有四个，最下面往下拉，才可以看到第五个。
这些功能板块围绕一个最核心的功能展开。这个最核心的功能叫“根据用户类型呈现内容。”
每一个用户在进入这个论坛时，他的身份默认是游客
在注册账号成功登录后，再次打开这个论坛，用户身份就是后面那四类
每一个用户注册账号后，他可以通过验证他的具体身份、具体能力、具体经济实力、是不是博主，来开启不同的用户类型身份。
在不同的用户身份类型情况下进入这个论坛，这个用户可以相应的在不同的功能板块获取到他所需要的信息。
举例：一个认证了投资者身份的用户，他可以在项目宣传板块看到正在寻求投资的项目宣传书。
但是如果没有认证投资者身份的用户，他即便可以看到一个正在宣发的项目，但是他是看不到这个项目正在融资的这部分信息的。
同理，一个认证的施工者身份的用户，他可以看到正在寻找施工（或是做模型、或是做建筑、或是当策划这剧情等等）伙伴的项目书。
这便是这个论坛最核心的基础功能，即“根据用户类型呈现内容。
不同类型的用户会开启不同的功能，鉴赏家是博主，也就是有能力把项目成品最终输出到自媒体渠道的人。

第二个事业大方向是组织革新。联创的平台本质上也是一种组织形式，跟线下的组织形式不同，线下的组织形式是现实中一个办公地点，通过招聘，把相应能力的人，能够解决线下的这个公司或这个企业成立的目标的相应能力的人聚到一起之后，然后去开展业务，去达成这个企业的目标，完成业务。

与这种线下的组织形式所不同的是，联合创作平台，它是把招聘，还有开展业务办公以及最终项目落地要做的一个利益分配，这些线下企业的各个必要的功能部分，完全搬到线上的一个差别。

先说线下的组织形式和联合创作这个组织形式，它们在招聘上的不同，首先线下的我们去公司要去一个单位去上班儿，想去做事情，对接HR，无论是网络上招聘，反正都要经过线下的面试，而面试的这个部分，HR招工招聘的这个部分，在联创的平台里面，就是大家网络上根据通过互联网信用体系，也就是对用户的一个更真实的身份，更完整的一个身份情况进行一个认证，这是联合创的平台这种组织形跟线下的一般组织形式之间的在招聘、在面试的这个功能部分上的一个替换，也就是线上的互联网信用体系的一个更真实更完整的一个对用户身份的认证同等于线下的一个招聘、面试的过程，这两个功能部分是一样的。

然后是组织形式上的，在办工开展工作的这个功能部分的差别，先说线下的，线下的一般企业的工作办公的这个功能部分，就是大家线下聚到一起，然后在各自工位，各自的部门，各自的领导领导下去一起去开展工作，跟同事一起做事。而在联创的平台里面，大家在首先都是在网络上，然后在相应的你们共同组建起的，联合创作团队里面去开展工作，都是在网络上，这是线上和线下的组织形式之间在办公的这个功能部分上的不同。联合创作平台就是在网络上，完全把办公功能搬到网络上。

最后是各种组织形式的一个最后的一个功能部分，也就是在开展业务的过程中，以及业务在产出效益跟企业运营的过程中，涉及到那些利益如何分配。

在线下的企业里面，我们在里面做事，会有开工资，劳务合同等等的东西用于进行利益分配，然后完成一个项目的时候有分成什么的，这些都体现在合同上。

而在线上的组织形式平台里面，用户们聚到一起，在联合创作平台里，他们在组建的创作团队里面完成一个项目后，或者开展工作的过程中或开展之前，也会跟我们平台去沟通这个利益该如何分配、怎样分配的问题，这个功能部分也是完全搬到线上的，也就是将来我们会提供给那一个个在我们平台内组建联合创作团队的那些团队们的，咨询利益该如何分配的服务，在线上建立利益分配合同的服务。

总而言之，组织形式革新的最终结果是把线下组织形式，也就是这种开展各种业务的，各个行业、各个领域的这些企业们各个功能部分完全搬到线上。
